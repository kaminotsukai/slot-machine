# プロダクト概要

## スロットマシンアプリケーション

TypeScriptで構築されたブラウザベースのスロットマシンゲームアプリケーション。

### 主要機能

- **3リールスロットマシン**: ランダムシンボル生成を備えた3つの独立したリール
- **個別停止機能**: 各リールを個別に停止できる目押し機能
- **勝敗判定**: シンボルの組み合わせに基づく自動勝敗評価
- **視覚的フィードバック**: スムーズなアニメーションと直感的なUI
- **レスポンシブデザイン**: さまざまな画面サイズに対応

### アーキテクチャ原則

- **MVC パターン**: Model-View-Controller アーキテクチャに基づく明確な関心の分離
- **コンポーネント分離**: 各コンポーネントは単一責任を持つ
- **型安全性**: TypeScript の厳格な型チェックを活用
- **テスト駆動**: ユニットテストとプロパティベーステストの両方を実装

### コアコンポーネント

- **GameEngine**: ゲームフローの統括とオーケストレーション
- **StateManager**: ゲーム状態の遷移管理
- **ReelManager**: リールの回転とシンボル生成
- **WinEvaluator**: 勝敗条件の判定
- **UserInterface**: UI レンダリングとユーザーインタラクション
